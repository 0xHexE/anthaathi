Required Tools

- [Gradle](https://gradle.org/install/)
- [Node](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/getting-started/install)

### Run Development for CRM

```bash
yarn
./gradlew :apps:anthaathi-crm-web-client:runDev
```
